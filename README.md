# School IERP

## Overview
School IERP (Integrated Education Resource Planning) is a comprehensive solution designed to manage and streamline the administrative and academic processes of educational institutions. It provides a centralized platform for managing student information, attendance, grades, timetables, fees, library resources, and staff information.

## Features
- **Student Information Management**: Maintain detailed records of student information.
- **Attendance Tracking**: Track and manage student attendance efficiently.
- **Grade Management**: Record and manage student grades and academic performance.
- **Timetable Scheduling**: Create and manage class schedules and timetables.
- **Fee Management**: Handle fee collection and manage financial records.
- **Library Management**: Manage library resources and track book lending.
- **Staff Management**: Maintain staff records and manage payroll.

## Technologies Used
- **Java**: Core programming language used for development.
- **Apache Tomcat**: Web server and servlet container.
- **MySQL**: Database management system for storing application data.

## Requirements
- **Java Development Kit (JDK)**: Version 17 or higher.
- **Apache Tomcat**: Version 10.0.27 or higher.
- **MySQL**: Version 8.0 or higher.
- **Eclipse IDE**: Recommended for development.

## Setup Instructions
1. **Clone the repository**:
    ```sh
    git clone https://github.com/yourusername/School-IERP.git
    ```
2. **Import the project** into your IDE (Eclipse recommended).
3. **Configure the Tomcat server** in your IDE.
4. **Update the MySQL connector library path** in the `.classpath` file if necessary.
5. **Run the project** on the server.

## Project Structure
- `src/main/java`: Contains the Java source code.
- `src/main/resources`: Contains the resource files.
- `src/main/webapp`: Contains the web application files.
- `build/classes`: Output directory for compiled classes.
- `.classpath`: Eclipse classpath configuration file.
- `.project`: Eclipse project configuration file.

## Usage
1. **Login**: Use the provided credentials to log in to the system.
2. **Dashboard**: Access the dashboard to view an overview of the system.
3. **Manage Students**: Add, update, or delete student information.
4. **Track Attendance**: Record and view student attendance.
5. **Manage Grades**: Enter and view student grades.
6. **Schedule Timetables**: Create and manage class schedules.
7. **Manage Fees**: Record and track fee payments.
8. **Library Management**: Manage library resources and track book lending.
9. **Staff Management**: Add, update, or delete staff information and manage payroll.
